# Cálculo de Frete

Projeto para calcular custos de frete com base em informações como nota fiscal, peso e outras variáveis.

## Estrutura de Pastas


## Como Usar

1. Abra o arquivo `index.html` em seu navegador.
2. Preencha os campos do formulário e clique em "Calcular Frete".
3. Os resultados serão exibidos com um efeito de flip.

## Dependências

- [Bootstrap 5.3.0](https://getbootstrap.com/)
- [jQuery 3.6.0](https://jquery.com/)
- [jQuery Mask Plugin 1.14.16](https://igorescobar.github.io/jQuery-Mask-Plugin/)
